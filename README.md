Завантажувач відео з YouTube та аудіо з SoundCloud

## Встановлення та запуск

Відкрий термінал і вводь по одній команді, після кожної натискай Enter:

**1. Скачати проект (або руками з сайту)**
```
git clone https://github.com/bestmoments/downloader
```

**2. Зайти в папку (або руками)**
```
cd downloader
```

**3. Встановити залежності (один раз — найважливіше)**
```
bash setup.sh
```

**4. Запустити**
```
python3 downloader.py
```
