Завантажувач відео та аудіо з YouTube і SoundCloud

## Встановлення та запуск

Відкрий термінал і вводь по одній команді, після кожної натискай Enter:

**1. Скачати проект (або руками з сайту)**
```
git clone https://github.com/bestmoments/downloader
```

**2. Зайти в папку (або руками)**
```
cd downloader
```

**3. Встановити залежності (один раз — найважливіше)**
```
bash setup.sh
```

**4. Запустити**

Або просто програма:
```
python3 downloader.py
```

Або Веб інтерфейс:
```
python3 web.py
```

При запуску сервер сам покаже адресу для інших пристроїв у мережі:
```
 * Пристрої в мережі:  http://<ім'я-твого-mac>.local:8080
```
Відкрий цю адресу у браузері на телефоні або іншому пристрої в тому ж Wi-Fi.

> Дізнатись ім'я свого Mac: **Системні налаштування → Загальні → Про цей Mac → Ім'я**
